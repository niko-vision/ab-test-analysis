# A/B Test Analysis

This portfolio project analyzes A/B test results for an online store and evaluates whether differences between the control and experimental groups are statistically significant.

## Objective

To compare the control and experimental groups and determine whether the observed differences in key conversion metrics are statistically significant.

## Metrics

- add_payment_info / session
- add_shipping_info / session
- begin_checkout / session
- new_accounts / session

## Files

- ab_test_analysis.ipynb — notebook with SQL extraction, Python analysis, statistical testing, and conclusions
